🚨 Cómo Reportar y Bloquear el Ciberacoso

Guía detallada paso a paso para reportar usuarios maliciosos y bloquear acosadores en cada plataforma. Sigue las instrucciones al pie de la letra para protegerte a ti y a tus hijos.

⚠️ Si hay peligro inmediato, llama al 911 (o 104 en Panamá)

Las amenazas de muerte, violencia física o acoso sexual son delitos. No esperes, contacta a las autoridades de inmediato.

📸 Antes de reportar o bloquear: Toma capturas de pantalla de TODOS los mensajes, comentarios o publicaciones ofensivas. Guarda las fechas, horas y URLs. Esta evidencia es crucial si necesitas escalar el caso a las autoridades o a la escuela.
📸
Instagram
Reportar una publicación o comentario: Presiona los tres puntos (⋮) en la esquina superior derecha de la publicación → Reportar → Selecciona el motivo (Acoso o intimidación, Contenido inapropiado, etc.) → Sigue las instrucciones.
📍 Ruta: Publicación → ⋮ → Reportar → Acoso o intimidación
Reportar un mensaje directo: Abre el chat → Presiona el nombre del usuario arriba → Reportar → Selecciona el motivo. También puedes mantener presionado el mensaje específico y seleccionar "Reportar".
📍 Ruta: Chat → Nombre usuario → Reportar
Bloquear a un usuario: Ve al perfil del acosador → Presiona los tres puntos (⋮) arriba a la derecha → Bloquear → Confirma. También puedes Restringir (los comentarios del usuario restringido solo los ve él).
📍 Ruta: Perfil → ⋮ → Bloquear
Configuración de privacidad: Perfil → ⋮ → Configuración → Privacidad → Cuenta privada (activar). También puedes: Desactivar comentarios en publicaciones específicas, ocultar historias a personas específicas, y revisar etiquetas antes de que aparezcan.
👤
Facebook
Reportar una publicación o comentario: Presiona los tres puntos (⋮) en la parte superior derecha de la publicación → Buscar ayuda o reportar publicación → Selecciona "Acoso" o "Bullying" → Sigue las instrucciones.
📍 Ruta: Publicación → ⋮ → Reportar publicación → Acoso
Reportar un perfil falso: Ve al perfil → Presiona ⋮ debajo de la foto de portada → Buscar ayuda o reportar perfil → Selecciona "Cuenta falsa" o "Suplantación de identidad".
Bloquear a un usuario: Ve al perfil → Presiona ⋮ → Bloquear → Confirma. También puedes desde: Configuración → Bloqueos → Agregar a lista de bloqueados.
📍 Ruta: Perfil → ⋮ → Bloquear
Configuración de seguridad: Configuración → Seguridad e inicio de sesión → Activar Autenticación en dos pasos (2FA). También: Revisa sesiones activas, activa alertas de inicio de sesión no reconocido.
𝕏
Twitter / X
Reportar un tweet: Presiona los tres puntos (⋮) en el tweet → Reportar tweet → Selecciona "Es abusivo o dañino" → "Acoso dirigido a mí" o "Acoso dirigido a alguien más".
📍 Ruta: Tweet → ⋮ → Reportar → Acoso
Reportar un mensaje directo: Abre el DM → Presiona ⋮ → Reportar mensaje → Selecciona el motivo.
Bloquear a un usuario: Ve al perfil → Presiona ⋮ → Bloquear → Confirma. También puedes Silenciar (no ver sus tweets sin que ellos lo sepan) o Reportar directamente.
📍 Ruta: Perfil → ⋮ → Bloquear
Configuración de privacidad: Más → Configuración y privacidad → Privacidad y seguridad → Proteger tus tweets (cuenta privada). También: Desactiva la ubicación en tweets, limita quién puede enviarte DMs, silencia palabras específicas.
💬
WhatsApp
Reportar un contacto o número desconocido: Abre el chat → Presiona ⋮ (los tres puntos) → Más → Reportar → Selecciona "Reportar y bloquear" o solo "Reportar". WhatsApp recibirá los últimos mensajes para revisión.
📍 Ruta: Chat → ⋮ → Más → Reportar → Reportar y bloquear
Bloquear a un contacto: Abre el chat → Presiona ⋮ → Más → Bloquear → Confirma. También desde: Configuración → Cuenta → Privacidad → Contactos bloqueados → Agregar.
📍 Ruta: Chat → ⋮ → Más → Bloquear
Configuración de privacidad: Configuración → Cuenta → Privacidad: Quién puede ver mi foto de perfil (selecciona "Mis contactos" o "Nadie"), Última vez (elige quién la ve), Confirmaciones de lectura (desactivar si prefieres), Llamadas (silenciar llamadas de números desconocidos).
🎵
TikTok
Reportar un video o comentario: En el video, mantén presionado o presiona ⋮ → Reportar → Selecciona "Acoso o bullying" → Sigue las instrucciones. Para comentarios: mantén presionado el comentario → Reportar.
📍 Ruta: Video → ⋮ → Reportar → Acoso o bullying
Reportar un mensaje directo: Abre el chat → Presiona ⋮ → Reportar → Selecciona el motivo.
Bloquear a un usuario: Ve al perfil → Presiona ⋮ (arriba a la derecha) → Bloquear → Confirma. También puedes Restringir para limitar la interacción sin bloquear completamente.
📍 Ruta: Perfil → ⋮ → Bloquear
Configuración de privacidad: Perfil → ⋮ → Configuración y privacidad → Privacidad: Cuenta privada (activar), Quién puede enviarte mensajes, Quién puede comentar tus videos, Filtro de comentarios (activar filtro de palabras clave y spam).
▶️
YouTube
Reportar un comentario: Junto al comentario, presiona los tres puntos (⋮) → Reportar → Selecciona "Acoso o bullying" → Envía.
📍 Ruta: Comentario → ⋮ → Reportar → Acoso o bullying
Reportar un video: Debajo del video, presiona ⋮ → Reportar → Selecciona "Contenido dañino o peligroso" → "Acoso o ciberacoso".
Bloquear a un usuario en YouTube: Ve al canal del usuario → Acerca de → Bandera (🚩) → Bloquear usuario. También desde: Configuración → Avanzado → Usuarios bloqueados.
📍 Ruta: Canal → Acerca de → 🚩 → Bloquear usuario
Configuración de seguridad: Configuración → Privacidad: activa Modo restringido (filtra contenido potencialmente inapropiado), Pausar historial, Gestionar comentarios (revisar retenidos, aprobar o eliminar). Para padres: activa YouTube Kids o Supervisado por padres.
